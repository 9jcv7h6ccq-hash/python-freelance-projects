# Python Automation & Data Tools

This repository contains a collection of practical Python scripts focused on automation, data cleaning, and data extraction.  
All projects are designed to solve real-world problems and can be easily customized for different use cases.

---

## 1. File Organizer Script

### Description
A Python script that automatically organizes files in a selected directory based on their file types (e.g. Images, Documents, Videos).

### Features
- Sorts files into folders by extension
- Reduces manual file management
- Simple and fast execution

### Technologies
- Python 3
- os
- shutil

### Usage
```bash
python file_organizer.py
```
